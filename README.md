<h1 align="center">eXtHealth</h1>
eXtHealth is a browser extension that fact-checks health claims on X posts and provides timely health tips and reminders.


<h2>Local Setup & Installation</h2>  

development:
```
npm run start
```

production:
```
npm run build
```
